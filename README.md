# kickstart-modular.nvim

Personal Neovim configuration based on [kickstart-modular.nvim](https://github.com/dam9000/kickstart-modular.nvim).

## Quick Start

```bash
git clone https://github.com/taasaa/kickstart-modular.nvim.git ~/.config/nvim
cd ~/.config/nvim
git remote add upstream https://github.com/dam9000/kickstart-modular.nvim.git
nvim
```

## Documentation

See [SETUP.md](./SETUP.md) for full documentation.

## Features

- **Leader:** `,`
- **Theme:** Tomorrow Night Bright
- **Plugins:** neo-tree, telescope, LSP, blink.cmp, treesitter, trouble, vim-surround, vim-repeat, render-markdown

## Updates

```bash
git fetch upstream
git merge upstream/master
git push
```
